# Face Detection System Using OpenCV

## Project Overview

This project implements a real-time Face Detection System using Python and OpenCV. The application captures video from a webcam, detects human faces using Haar Cascade Classifiers, and draws bounding boxes around detected faces.

The system can detect multiple faces simultaneously and display the detection results in real time.

## Features

* Real-time face detection using webcam
* Face detection using OpenCV Haar Cascades
* Bounding boxes around detected faces
* Multiple face detection support
* Simple and user-friendly implementation
* Press 'Q' to exit the application

## Technologies Used

* Python
* OpenCV
* NumPy

1. Clone the repository

2. Install required libraries

pip install -r requirements.txt

3. Run the project

python face_detection.py

## How It Works

1. The webcam captures live video frames.
2. Each frame is converted to grayscale.
3. Haar Cascade Classifier detects faces in the frame.
4. Bounding boxes are drawn around detected faces.
5. The processed video is displayed in real time.

## Output

* Detects faces from webcam input.
* Displays a green rectangle around detected faces.
* Supports detection of multiple faces simultaneously.
